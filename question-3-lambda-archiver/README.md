# Question 3: Lambda Function for Scheduled Data Archiving

Automated Lambda function that archives DynamoDB items older than 30 days to S3.

## Quick Start
```bash
npm install
npm run deploy
```

## Features

- Daily automated archival
- Batch processing
- S3 lifecycle management
- Comprehensive error handling

## Testing
```bash
npm run invoke
npm run logs
```
